# gorm-yasdb|崖山数据库GORM驱动

gorm-yasdb 是 YashanDB 的 GORM 驱动，基于 GORM 框架实现，依赖崖山数据库 Go 驱动。

### 特性

- ✅ 兼容 GORM v1.x
- ✅ 自动迁移（AutoMigrate）
- ✅ 完整的 CRUD 操作支持
- ✅ 事务支持
- ✅ 关联关系（HasOne, HasMany, BelongsTo, Many2Many）
- ✅ 钩子与回调
- ✅ 连接池支持
